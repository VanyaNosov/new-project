What does Fortify do ?
Use your smart card or security token
that has already been enrolled with a
certificate with web applications.
Enroll for a new certificate or renew an
existing one.
Build applications that can sign/verify or
encrypt/decrypt using locally installed
certificates, smart cards or security
tokens.
