Frequently asked questions
How does Fortify work? Fortify is a locally installed application that listens on a known TCP port. Web
applications use the Fortify WebCrypto polyfill to communicate with this application
which enables the web application to use smart cards, security tokens and locally
installed certificates.
What smart cards do you support? Generally Fortify should work with any PKCS#11 enabled smart card. For it to work
though it needs to know which driver to load for a given smart card.
